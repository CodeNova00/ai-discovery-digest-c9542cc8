## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **AI**: OpenAI / Hugging Face Transformers  
- **Email Service**: NodeMailer  
- **Deployment**: Vercel / Render / MongoDB Atlas
